# Contract-Consent-Agent

## Run Tests Independently by Agent

### Contract Agent
```bash
pnpm test-cca-contract
```

### Consent Agent
```bash
pnpm test-cca-consent
```

## Generate Test Reports Independently by Agent

### Contract Agent
```bash
pnpm report-cca-contract
```

### Consent Agent
```bash
pnpm report-cca-consent
```

## Run Full Test Suite
```bash
pnpm test
```

## Build the Project
```bash
pnpm build
```
